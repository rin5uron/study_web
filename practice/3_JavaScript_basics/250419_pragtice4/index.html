DOM操作の基本

DOMとは？

DOM（Document Object Model）は、ウェブページを木（ツリー）のような構造で表したものです。
PCはHTMLを読み込む時、各要素（タグ）を木の枝のように繋がっています。

DOMは、JavaScriptでWebページの要素を操作するための仕組みです。

DOM操作の4つのステップ

ステップ 説明 課題
1. 要素の取得

まずは、ページの中から欲しい要素を見つけます。

例：IDで要素を見つける

let 要素 = document.getElementById("要素のID");
2. 要素の内容変更

見つけた要素の中身（テキスト）を変更します。

例：テキストを変更する

要素.textContent = "新しいテキスト";
3. スタイルの変更

要素の見た目（色や大きさなど）を変更します。

例：背景色を変更する

要素.style.backgroundColor = "red";
4. 要素の追加と削除

新しい要素を作ったり、いらない要素を削除したりします。

例：新しい要素を作成して追加する

let 新しい要素 = document.createElement("p");
新しい要素.textContent = "新しい段落です";
親要素.appendChild(新しい要素);
