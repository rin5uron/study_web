DOM操作の基本

DOMとは？

DOM（Document Object Model）は、ウェブページを木（ツリー）のような構造で表したもの。
PCはHTMLを読み込む時、各要素（タグ）を木の枝のように繋げる。

DOMとは
JavaScriptでWebページの要素を動かす仕組み

DOM操作の4つのステップ

1
要素の取得

まずは、ページの中から欲しい要素を見つける。

例：IDで要素を見つける

let 要素 = document.getElementById("要素のID");
練習問題
2
要素の内容変更

見つけた要素の中身（テキスト）を変更する。

例：テキストを変更する

要素.textContent = "新しいテキスト";
練習問題
3
スタイルの変更

要素の見た目（色や大きさなど）を変更する。

例：背景色を変更する

要素.style.backgroundColor = "red";
練習問題
4
要素の追加と削除

要素を新しく作ったり削除する

例：新しい要素を作成して追加する

let 新しい要素 = document.createElement("p");
新しい要素.textContent = "新しい段落";
親要素.appendChild(新しい要素);
練習問題
