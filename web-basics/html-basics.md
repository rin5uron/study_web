## HTMLとは
HTMLは「HyperText Markup Language」の略で、Webページの骨組みを作るための言語です。
テキスト、画像、リンクなどの要素を「タグ」を使って構造化します。

## 基本的な構造
HTMLファイルの基本構造は以下のようになっています：

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>ページのタイトル</title>
</head>
<body>
    <!-- ここに表示したい内容を書きます -->
    <h1>こんにちは！</h1>
    <p>これは段落です。</p>
</body>
</html>
```

### 構造の説明
- `<!DOCTYPE html>`: HTMLファイルであることを宣言
- `<html>`: HTMLの開始と終了
- `<head>`: ページの設定情報
- `<body>`: 実際に表示される内容

## 主要なタグ
よく使用される基本的なタグを紹介します：

### 見出し
```html
<h1>最も大きい見出し</h1>
<h2>2番目に大きい見出し</h2>
<h3>3番目に大きい見出し</h3>
```

### 段落とテキスト
```html
<p>これは段落です。</p>
<strong>太字のテキスト</strong>
<em>強調されたテキスト</em>
```

### リスト
```html
<!-- 番号付きリスト -->
<ol>
    <li>最初の項目</li>
    <li>2番目の項目</li>
</ol>

<!-- 箇条書きリスト -->
<ul>
    <li>項目1</li>
    <li>項目2</li>
</ul>
```

#### リストタグの使い分け
- `<ul>`: Unordered List（順序なしリスト）。箇条書きに使用。項目の順番に意味がない場合に使用。
- `<ol>`: Ordered List（順序ありリスト）。番号付きリストに使用。項目の順番に意味がある場合に使用。
- `<li>`: List Item（リスト項目）。`<ul>`または`<ol>`の中で各項目を定義するために使用。

### リンクと画像
```html
<a href="https://www.example.com">リンクテキスト</a>
<img src="画像のパス" alt="画像の説明">
```

### セマンティックタグ
```html
<!-- ヘッダー：ページの上部に表示される導入エリア -->
<header>
    <h1>サイトタイトル</h1>
    <nav>
        <ul>
            <li><a href="#">ホーム</a></li>
            <li><a href="#">お問い合わせ</a></li>
        </ul>
    </nav>
</header>

<!-- メインコンテンツ -->
<main>
    <article>メインの記事</article>
</main>

<!-- フッター：ページの下部エリア -->
<footer>
    <p>&copy; 2025 私のウェブサイト</p>
</footer>
```

## HTMLの重要要素

### 文字コード指定
HTMLファイルでは、文字コードの指定が非常に重要です。文字コードはコンピュータが文字を扱うための数値表現システムで、適切に指定しないとブラウザで文字化けが発生する可能性があります。

```html
<head>
    <meta charset="UTF-8">
    <!-- ↑この行で文字コードをUTF-8と指定しています -->
</head>
```

#### 主な文字コード
- **UTF-8**: 現在の国際標準。すべての言語（日本語含む）と絵文字をサポート
- **Shift-JIS**: 日本語Windows環境で使われてきた文字コード
- **EUC-JP**: 日本語Unix/Linux環境で使われてきた文字コード
- **ASCII**: 基本的な英数字のみをサポートする最も古い形式

**重要**: 現代のWeb開発では、UTF-8を使用することが標準的です。この指定がないと、ブラウザによっては文字化けを起こす可能性があります。

### タイトルタグ
`<title>`タグはHTMLドキュメントにとって非常に重要な要素で、`<head>`セクション内に記述します。一言で言えば「Webページの名前を定義し、ブラウザ、検索エンジン、ユーザーがページを識別するための最重要メタデータ」です。

```html
<head>
    <meta charset="UTF-8">
    <title>ページタイトル | サイト名</title>
</head>
```

#### タイトルタグの役割
1. **ブラウザ表示**: ブラウザのタブやウィンドウ上部に表示されるテキスト
2. **ブックマーク**: ユーザーがページをブックマークした際の名前として使用される
3. **検索エンジン最適化(SEO)**: 検索結果に表示されるタイトルとなり、検索順位に影響
4. **アクセシビリティ**: スクリーンリーダーがページを識別する際に使用される

#### 良いタイトルの書き方
- ページの内容を的確に表す（40-60文字程度）
- 重要なキーワードを含める
- ユニークなタイトルにする（サイト内で重複しない）
- サイト名を含める（例: 「ページ内容 | サイト名」）

**注意**: 一つのHTMLドキュメントには必ず1つのタイトルタグを設定する必要があります。

## 実践例
シンプルな自己紹介ページを作ってみましょう：

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>自己紹介</title>
</head>
<body>
    <h1>私について</h1>
    
    <h2>基本情報</h2>
    <ul>
        <li>名前：山田太郎</li>
        <li>趣味：プログラミング、読書</li>
    </ul>

    <h2>好きな食べ物</h2>
    <ol>
        <li>ラーメン</li>
        <li>カレー</li>
        <li>寿司</li>
    </ol>

    <p>よろしくお願いします！</p>
</body>
</html>
```

## 練習課題
1. 上記の自己紹介ページを参考に、自分の自己紹介ページを作成してみましょう。
2. 以下の要素を必ず含めてください：
   - 見出し（h1, h2）
   - 段落（p）
   - リスト（ul または ol）
   - 太字のテキスト（strong）
   - リンク（a）

## 📕 専門用語集

| 用語 | 説明 |
|------|------|
| HTML | HyperText Markup Languageの略。Webページの構造を定義するためのマークアップ言語 |
| タグ | HTMLの基本要素。<>で囲まれた命令で、開始タグと終了タグがある（例：`<p></p>`） |
| 要素 | 開始タグ、内容、終了タグからなるHTMLの構成単位（例：`<p>これは段落です</p>`） |
| 属性 | タグに追加情報を与えるもの（例：`<a href="URL">` の `href` が属性） |

## 主要なHTMLタグ一覧

| タグ | 説明 | 使用例 | 重要度 |
|------|------|--------|--------|
| `<!DOCTYPE html>` | HTMLバージョンの宣言 | ファイルの先頭に配置 | 最重要 |
| `<html>` | HTML文書の開始と終了 | `<html>...</html>` | 最重要 |
| `<head>` | ページの設定情報を格納する領域 | `<head><title>...</title></head>` | 最重要 |
| `<meta>` | メタデータ（文字コードなど）を指定 | `<meta charset="UTF-8">` | 最重要 |
| `<title>` | ページのタイトルを定義 | `<title>ページタイトル</title>` | 最重要 |
| `<body>` | ブラウザに表示されるコンテンツを格納 | `<body>...</body>` | 最重要 |
| `<header>` | ページ上部の導入エリア | `<header>...</header>` | 高 |
| `<nav>` | ナビゲーションリンクを含む領域 | `<nav><ul><li>...</li></ul></nav>` | 高 |
| `<main>` | ページの主要コンテンツ（1ページに1つ） | `<main>...</main>` | 高 |
| `<section>` | 関連するコンテンツのまとまり | `<section><h2>...</h2>...</section>` | 高 |
| `<article>` | 独立して完結するコンテンツ | `<article><h2>...</h2>...</article>` | 高 |
| `<footer>` | ページ下部のフッター領域 | `<footer>...</footer>` | 高 |
| `<h1>` - `<h6>` | 見出し（重要度順） | `<h1>最大見出し</h1>` | 高 |
| `<p>` | 段落 | `<p>これは段落です</p>` | 高 |
| `<a>` | リンク | `<a href="URL">テキスト</a>` | 高 |
| `<img>` | 画像 | `<img src="image.jpg" alt="説明">` | 高 |
| `<ul>` | 順序なしリスト（箇条書き） | `<ul><li>項目</li></ul>` | 中 |
| `<ol>` | 順序ありリスト（番号付き） | `<ol><li>項目</li></ol>` | 中 |
| `<li>` | リスト項目 | `<li>リストの項目</li>` | 中 |
| `<div>` | 汎用ブロック要素 | `<div>...</div>` | 中 |
| `<span>` | 汎用インライン要素 | `<span>...</span>` | 中 |
| `<strong>` | 強調（太字） | `<strong>重要</strong>` | 中 |
| `<em>` | 強調（斜体） | `<em>強調</em>` | 中 |
| `<br>` | 改行 | `テキスト<br>改行` | 中 |
| `<hr>` | 水平線 | `<hr>` | 低 |
| `<!--...-->` | コメント | `<!-- これはコメントです -->` | -- |

### sectionとarticleの違い
- `<section>`: 関連するコンテンツをグループ化するためのタグ。通常見出しを含む。
- `<article>`: 独立して完結するコンテンツを表すタグ。それ単体で意味をなす自己完結的な内容。

使い分け：
- そのコンテンツが**独立して配信・再利用できる**なら → `<article>`
- 単に関連するコンテンツを**グループ化したい**だけなら → `<section>`
